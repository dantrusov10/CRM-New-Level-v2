# PocketBase schema files

Что здесь оставлено:
- `collections.json` — рабочая коллекционная схема для bootstrap/init.
- `pb_schema.json` — актуальная схема из репозитория.
- `init.sh` / `run.sh` — штатные скрипты запуска.

Что убрано:
- старые stage-патчи и backup-файлы, которые путали импорт и создавали риск случайно применить не тот JSON.

Важно:
- для обновления живой PocketBase через UI используйте только свежий export из самой админки этой же инстанции,
  а не исторические json-файлы из репозитория.
